# Employment and Data Safety Statement

This repository supports a public-dataset-based academic manuscript. It does not contain
employer data, Salesforce data, customer records, confidential CRM configurations, internal
system information, proprietary implementation details, commercial deployment metrics,
API keys, private screenshots, or employer-owned source code.

All experiments are based only on publicly available datasets: MS MARCO, MSDialog, and WixQA.

The architecture descriptions are generalized and are intended to reflect public knowledge
of enterprise software engineering, retrieval-augmented generation, and CRM-style information
systems. They do not disclose or reproduce any internal system, private architecture,
confidential configuration, non-public metric, or employer-specific implementation.
