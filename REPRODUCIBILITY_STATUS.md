# Reproducibility Status

This repository provides reviewer-accessible reproducibility materials for the manuscript
"A Multi-Source Retrieval-Augmented Generation Framework for Intelligent Agent Orchestration
in Enterprise CRM Systems."

The repository includes:

* retrieval and evaluation scripts;
* configuration files;
* summary result files aligned with the manuscript tables;
* annotation guidelines and anonymized annotation sample files;
* documentation describing data availability and limitations.

The full public datasets are not redistributed in this repository. Users should download
MS MARCO, MSDialog, and WixQA from their official sources and configure local paths in
`configs/dataset_config.yaml`.

This repository does not contain proprietary data, employer data, customer records,
confidential CRM configurations, internal system information, commercial deployment metrics,
API keys, or private screenshots.
