# Data Availability Statement

All experiments in the associated paper use publicly available datasets.
No proprietary data, employer data, customer records, or confidential CRM
configuration data is used in any experiment or included in this repository.

## Publicly Available Datasets

| Dataset | License | Availability |
|---------|---------|-------------|
| MS MARCO Passage Ranking | MS MARCO License (non-commercial research) | https://microsoft.github.io/msmarco/ |
| MSDialog | CC BY 4.0 | https://ciir.cs.umass.edu/downloads/msdialog/ |
| WixQA | MIT | https://github.com/wix-incubator/WixQA |

## What Is NOT in This Repository

- Real customer queries or responses
- Employer or organizational data
- Internal CRM configuration (objects, fields, workflows)
- Internal performance metrics or operational data
- Any personally identifiable information (PII)
- Any data subject to non-disclosure obligations

## Annotation Data

Human annotation labels included in `annotations/anonymized_hallucination_labels_sample.csv`
were collected using queries derived from publicly available MS MARCO topics.
No real customer queries were used. Annotator identities are not disclosed.

## Full Annotation Set

The complete set of 500 annotation records is available from the corresponding author
(chitracrmexpert@gmail.com) upon reasonable request, subject to the MS MARCO license
terms under which the query topics were derived.
